# 产品支持页 UI 规格

## 页面任务

产品支持页负责展示“这些能力怎么支撑品牌把日常运营做起来”，是全站最接近产品展示页的一页。

## 视觉关键词

- 产品感
- 界面感
- 执行效率
- 日常运营工作台

## 页面结构

1. Hero  
左文右图，右侧放产品工作台主界面。

2. 入口承接段  
展示扫码入口和统一承接方式。

3. 会员连接段  
展示留资、入会、积分权益等界面。

4. 运营触达段  
展示节点活动、会员触达、复购提醒界面。

5. 结果回看段  
展示数据复盘和效果分析界面。

6. 证明区  
突出“产品能不能真正被团队用起来”。

7. CTA 区  
引导预约产品演示。

## 版式重点

- 这一页视觉密度可以比其他页高
- 多用浏览器窗口、手机界面、数据图层、工作台 UI
- 每段右侧图可以带轻设备框或轻卡片堆叠
- 不要做传统软件官网那种重深色仪表盘

## 交互与动效

- 顶部 progress bar
- UI 截图卡片滚动进入时淡入
- 界面卡片 hover 轻微上浮
- 小数据点位可做轻脉冲动效
