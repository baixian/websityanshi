# 商务咨询页 UI 规格

## 页面任务

商务咨询页负责转化，把“先聊一次能得到什么”讲清楚，并让表单成为页面焦点。

## 视觉关键词

- 低门槛
- 明确
- 顾问感
- 转化导向

## 页面结构

1. Hero  
讲“先把方向和起点看清楚，再决定是不是现在开始”。

2. 适合来沟通的人群段  
帮助访客判断是否值得现在联系。

3. 一次沟通能得到什么段  
讲清咨询产出。

4. 轻量起步段  
讲“可以先从一个重点场景开始”。

5. 沟通后下一步段  
讲清是否适合做、先做什么、怎么开始。

6. 表单主区  
整页最强视觉焦点。

7. 信任补充区  
可放简短流程说明、顾问响应方式或保障说明。

## 版式重点

- 页面整体可以比其他页更短
- Hero 和表单区的视觉权重最高
- 表单区建议用 `左说明 + 右表单` 或 `上说明 + 下表单`
- 表单外层卡片要明显高于其他卡片层级

## 交互与动效

- 顶部 progress bar
- 表单输入 focus 状态清晰
- 按钮 hover 强于全站平均值
- 信任补充区做轻 reveal
