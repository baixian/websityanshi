# 合作方式页 UI 规格

## 页面任务

合作方式页负责让访客理解“怎么合作、适合哪种合作节奏、什么时候该从哪种方式开始”。

## 视觉关键词

- 选择感
- 节奏感
- 结构化
- 比咨询页更像方案对比页

## 页面结构

1. Hero  
说明“不是所有品牌都需要同一种合作方式”。

2. 诊断与试点合作段  
强调先判断、先验证。

3. 项目制合作段  
强调围绕新品、节点活动和动销项目集中发力。

4. 长周期陪跑合作段  
强调会员活跃、复购和长期运营。

5. 多渠道协同合作段  
强调多门店、多区域和多合作方协同。

6. 合作方式对比区  
做成 4 卡对比结构。

7. CTA 区  
引导预约合作沟通。

## 版式重点

- 上半段用长卷轴解释合作场景
- 下半段用对比卡结构收口
- 4 张合作卡片要有清晰层级，不做价格塔
- 推荐卡可略微放大或加更强渐变边

## 交互与动效

- 顶部 progress bar
- 对比卡 hover 提升
- 分段 reveal
- CTA 按钮 hover 上浮
