# 行业场景页 UI 规格

## 页面任务

行业场景页负责帮助访客快速对号入座，判断自己的业务更适合从什么场景开始。

## 视觉关键词

- 场景化
- 行业匹配
- 真实业务感
- 比产品页更少后台界面，更多消费品应用场景

## 页面结构

1. Hero  
说明“不同行业起点和重点不同”。

2. 快消与饮料场景段  
强调包装触点与终端动销。

3. 酒水与礼赠场景段  
强调节庆热度、礼赠场景和后续关系。

4. 食品与日化场景段  
强调高频复购和活跃经营。

5. 连锁零售与渠道场景段  
强调多门店、多区域和多终端联动。

6. 证明区  
说明行业不同，但核心问题接近。

7. CTA 区  
引导预约场景沟通。

## 版式重点

- 不做行业卡片墙
- 每个行业一整段，配一张代表场景图
- 场景图尽量带消费品、货架、门店、礼盒、终端物料等元素
- 每段可以加 2 到 3 个行业标签或典型关键词

## 交互与动效

- 顶部 progress bar
- 场景图 reveal
- 行业标签 hover 高亮
- CTA 区保持轻动画
