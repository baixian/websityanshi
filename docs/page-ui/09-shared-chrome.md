# 公共 Header / Footer UI 规格

## 文档目的

本文件用于解决当前首页和公共底部视觉脱节的问题，并定义后续全站统一的公共层改造方式。

## 使用说明

- 当前站点的问题审计请查看 [00-current-state-audit.md](/Users/wangwang/Desktop/扫码官网/website/docs/page-ui/00-current-state-audit.md)
- 本文件从这里开始只定义目标态的公共 Header / Footer 规格，不再混写现状问题说明

## 改造目标

1. 让 Header、Footer、品牌标识、导航、CTA 按钮进入同一套共享母版。
2. 让首页到页尾的颜色、留白、按钮、卡片、文字语气保持统一。
3. 让 Footer 不再只是信息堆叠，而是成为页面收口和信任建立的一部分。
4. 让这套公共层可以稳定推到 8 个页面，而不是首页单独特例。

## 总体风格方向

- 亮底产品展示官网
- 紫蓝渐变强调
- 浅灰白背景
- 轻卡片、轻描边、轻阴影
- 顶部克制、底部有层次但不过重

## Header 规格

### 页面角色

- 快速识别品牌
- 给出整站主导航
- 提供最高优先级 CTA
- 在滚动后保持清晰和轻量

### 视觉要求

- 背景以浅白或半透明浅底为主
- 滚动后增加轻描边、浅阴影和背景模糊
- Logo 区保持稳定，不随页面切换频繁变色
- 导航高亮以紫蓝渐变或高亮描边体现
- “商务咨询”按钮维持全站最高 CTA 层级

### 信息结构

1. 品牌标识
2. 主导航
3. 商务 CTA
4. 移动端菜单按钮

### 移动端规则

- 抽屉菜单沿用公共组件，不单独做首页特例
- 抽屉内 CTA 保持强按钮
- 抽屉下半部分保留商务联系信息，但减少冗余文字

## Footer 规格

### 页面角色

- 收口品牌价值
- 给出关键导航回路
- 建立信任感
- 承接联系与备案信息

### 不再使用的旧语法

- 暖棕色整段背景
- 偏旧站风格的整块文案堆叠
- 信息块之间缺少主次关系的并列排列

### 新 Footer 的视觉语法

- 以浅灰或浅紫白渐变收口
- 用细描边、浅底卡片、紫蓝高光统一风格
- 保留深色只作为点缀，不作为大面积主背景
- 与首页、解决方案页、产品页的按钮和卡片风格一致

### Footer 信息分层

#### 第一层：品牌与一句价值说明

- 品牌名
- 一句品牌价值概述
- 2 到 3 个小标签，说明适合的服务方向

#### 第二层：主导航与关键入口

- 先看什么
- 产品与合作
- 信任与咨询

#### 第三层：信任补充区

- 适合什么品牌
- 当前服务边界
- 首次沟通会先确认什么

#### 第四层：联系与法务信息

- 公司主体
- 咨询电话
- 服务时间
- 地址
- 备案与版权

### Footer 版式建议

1. 上半部分：`品牌区 + 导航区`
2. 中间部分：`3 张信任卡`
3. 下半部分：`法务与备案`

### Footer CTA 处理

- Footer 不单独做大面积强推销横幅
- 可在品牌区或导航区保留一个“预约方案沟通”按钮
- CTA 要比正文明显，但不能抢首页 CTA 的主位

## 首页验证策略

首页作为第一个验证页，重点检查以下 4 项：

1. 首屏结束到 Footer 之间，颜色和节奏是否断裂
2. Footer 按钮、标签和卡片是否和首页保持同一语法
3. Footer 的信息密度是否会把首页拉回旧站气质
4. 备案信息在视觉上是否足够克制，不抢主要内容

## 全站推广顺序

1. 先重做公共 Footer
2. 先接入首页验证
3. 再推到解决方案页和产品支持页
4. 再推到行业、案例、合作页
5. 最后推到安全保障页和商务咨询页

## 占位图与背景建议

公共层通常不需要复杂生图，但可以准备以下轻背景：

1. Header 背景高光纹理
- 用途：滚动后浅色模糊底的微氛围层
- 比例：`16:9`
- 推荐尺寸：`1920 × 1080px`
- 建议文件名：`shared-header-background.jpg`

2. Footer 品牌区背景
- 用途：品牌说明区轻氛围图
- 比例：`16:9`
- 推荐尺寸：`1920 × 1080px`
- 建议文件名：`shared-footer-brand-background.jpg`

3. Footer 信任卡背景纹理
- 用途：让信任卡不显得太平
- 比例：`4:3`
- 推荐尺寸：`1440 × 1080px`
- 建议文件名：`shared-footer-trust-texture.jpg`

## 背景 Prompt

### 1. Header 背景高光纹理

`明亮商业官网背景纹理，浅灰白背景，极轻紫蓝渐变高光，柔和模糊、极简、干净、有留白，适合作为浅色 Header 微背景，无文字，无水印`

### 2. Footer 品牌区背景

`现代品牌官网浅色背景，白色与浅紫灰渐变，带非常轻的弧形流线和柔和高光，优雅、克制、可信，适合作为 Footer 品牌说明区背景，无文字，无水印`

### 3. Footer 信任卡背景纹理

`浅色商务官网卡片背景纹理，白底带极轻紫蓝渐变边缘和柔和阴影感，简洁、理性、适合作为信任信息卡的轻纹理背景，无文字，无水印`

## 验收标准

改造完成后，应满足：

1. 首页到 Footer 不再像两个网站拼接
2. Header、Footer、按钮、品牌标识进入同一套视觉系统
3. Footer 信息分层清楚，既能收口又不显得沉重
4. 移动端抽屉、Footer 分组和备案信息都能清晰落地
