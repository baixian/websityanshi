# 客户案例页文案稿

## 页面定位

客户案例页负责建立信任，让访客看到问题、动作和结果是怎么连起来的。

## 建议案例成稿格式

每个案例最终建议统一为：

1. 品牌 / 项目名称
2. 场景说明
3. 原来卡在哪里
4. 关键动作怎么做
5. 最后结果怎么变

## 第一段：新品推广案例

- 标题：新品上市，要留下第一批人
- 副标题：很多新品在首波传播之后很快降温。更有价值的是把尝鲜用户接进后续触达，让热度继续沉淀成留资、活跃和再次购买。
- 收益点：
  - 首波活动后的留资空间更大
  - 尝鲜用户可以继续进入会员连接
  - 新品推广投入能留下长期价值

## 第二段：门店动销案例

- 标题：门店动销，不只看当次成交
- 副标题：门店活动最接近真实消费现场。除了当场核销，更要看用户离店之后品牌还能不能继续触达、继续带回购。
- 收益点：
  - 门店活动后的后续价值被放大
  - 线下触点后还有再沟通机会
  - 到店用户后续还有被带回购的空间

## 第三段：渠道活动案例

- 标题：渠道越多，越要可复制
- 副标题：当品牌同时面对不同区域、不同终端和不同合作伙伴时，案例最有参考价值的，是哪些动作可以复制、哪些结果能够稳定复现。
- 收益点：
  - 渠道动作可以统一复盘
  - 终端触点可以继续沉淀用户
  - 值得复制的方法可以沉淀下来

## 第四段：会员运营案例

- 标题：复购增长，要看平时怎么经营
- 副标题：能打动品牌的，往往不是某一次优惠带来多少成交，更在于品牌能否通过会员连接、内容触达和节点互动，把用户持续带回来。
- 收益点：
  - 会员关系更有活跃度和粘性
  - 后续触达更贴近用户购买节奏
  - 复购不只靠一次优惠带动

## 证明区

- 标题：相似场景，比大词更有说服力
- 副标题：比起只看做过什么，品牌更想知道相似场景值不值得做，做完会出现什么变化。

## 建议证据素材

- 每个案例至少补 1 个“原来问题”与 1 个“后来的变化”
- 每个案例尽量补 1 到 2 个结果指标：如 `留资提升 / 活跃提升 / 复购提升 / GMV 增长`
- 建议补品牌名、行业、项目周期和核心动作，避免案例只剩抽象结论
- 建议每个案例补 1 句“用户参与动作”：如 `扫码 / 签到 / 分享 / 积分 / 抽奖 / 兑换`
- 建议每个案例补 1 句“品牌提供的权益”：如 `礼包 / 积分 / 体验装 / 会员权益 / 活动资格`
